## Instalação
```bash
# Clone do Repositorio
$ git clone https://github.com/WesleyReboucas/Happy.git
```

### Web

```bash
# Na pasta web
$ cd Proffy/web

# Instalação de depedencias
$ yarn install

# Iniciando aplicação
$ yarn start
```
**Acesse:** http://localhost:3000/
